Fishpowered updated!

Here are some highlights and things from previous updates you may have missed…

Easily reach the top of a page with one hand

Desktop mode

Desktop mode is remembered for each site you enable it on. Note: some sites may use cookies to remember this regardless of the site setting.

Search engine shortcuts

Use Google for search accuracy but Duckduckgo for privacy?

Easily search using another search engine by using the first letter of it's name as a prefix e.g. "g search from google", "d search on duckduckgo" etc

Improvements to fav'ing, tagging, and downloading of videos

Sites make it difficult to extract the true video source so remember to be on the permanent link for the video i.e. if there are multiple videos on a page it probably won't work.

Bug fixes

Fixed switching to tabs occasionally not working/jumping back to previous tab, fixed cyrllic characters broken in search suggestions, fixed "Find in page" menu not always opening, crash fixes and more.

Twitch

To increase trust and transparency we've started live streaming development on Twitch. See the code, follow new features being developed, and watch us struggle to fix sneaky bugs. Unfortunately the developer can't concentrate when he knows he's being watched so he doesn't look at the chat though :P
